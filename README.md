# 실행

## 준비물

- Java 8 이상

## 소스 코드 받기

```bash
git clone https://github.com/kimjongmo/admin
```

## 소스 코드 빌드

```bash
cd admin
```

```bash
chmod +x ./gradlew
```

```bash
./gradlew build
```

## 실행

```bash
java -jar ./build/libs/admin.jar
```



# API 문서

어플리케이션 구동 후 http://localhost:9090/swagger-ui.html로 접속

각 컨트롤러 클릭시 해당 API의 정보를 볼 수 있다.
